Sistem Deteksi Katarak

Latih Model

Latih model SVM dengan optimisasi hyperparameter menggunakan gambar dari folder dataset.

Pastikan struktur dataset Anda:

dataset/train/normal/ - berisi gambar mata normal untuk training
dataset/train/cataract/ - berisi gambar mata katarak untuk training

*Sistem akan menguji berbagai split ratio (80:20, 70:30, 60:40) dan kernel SVM (Linear, Polynomial, RBF) untuk menemukan model terbaik.
